SmashNerd

SmashNerd is the quintessential site for fans of the timeless Smash Brothers Games. It compiles character and stage statistics into a clean and user-friendly site, allowing dedicated smash nerds to access all of the info they need to select the character that’s best for them.

Wheel of Fortune

Enjoy a round of this three-player recreation of the classic Wheel of Fortune game. Spin the wheel with friends and solve the puzzles to compete for a chance to spin the bonus wheel and take home massive prizes! Or fly solo and play against a couple of AI players!

A New Scope

Checkout your favorite starwars characters, planets, and vehicles in this space themed sight. Whether it's a 3d animated Millenium Falcon or the rotating galaxy menu, navigating this site is a pleasure. Built with the starwars API and fully tested with Jest/Enzyme

CSS aniMate

CSS Animate provides a playground for web developers to create and adjust CSS keyframes animations, displaying the results in real time. Find an animation you like, make your tweaks, and copy the code into your project. Create an account to make and save custom animations from scratch. 100% test coverage with Jest/Enzyme.
